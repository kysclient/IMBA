# IMBA 공식 홈페이지 및 관리자 시스템 구축 견적서

**작업일:** 2026년 2월 24일
**제안자:** 5년차 프리랜서 웹 개발자

---

## 1. 프로젝트 개요

- **목적:** IMBA(국제메디컬뷰티협회) 공식 홈페이지 프론트엔드 퍼블리싱(완료 단계)에 이은 백엔드 구축 및 데이터 연동, 통합 관리자(Admin) 페이지 개발
- **개발 스택 (예상):** Next.js(React), TypeScript, TailwindCSS, Node.js API, PostgreSQL(or MySQL), Prisma ORM, AWS S3(이미지 호스팅)
- **예상 소요 기간:** 약 5~6주

## 2. 작업 범위 (Scope of Work)

### A. 사용자 페이지 프론트엔드 고도화 및 API 연동 (기구축 퍼블리싱 기반)

1. **사용자 인증 (Auth)**
   - 로그인 / 회원가입 / 비밀번호 찾기 기능 연동
   - JWT 또는 세션 기반 권한 처리 (일반 회원 / 관리자)
2. **커뮤니티 및 갤러리 연동**
   - 공지사항, Q&A, 수강후기 게시판 CRUD(작성/수정/삭제/조회) 연동
   - 게시글 댓글, 조회수, 좋아요 기능 연동
   - 갤러리 이미지 AWS S3 업로드 및 리스팅 연동
3. **수강 신청 (Apply) 및 교육 과정 (Courses)**
   - 수강 신청 폼 데이터 DB 저장 및 알림(이메일 등) 연동
   - 동적 교육 과정 리스트 불러오기
4. **스토어 (Store)**
   - 상품 리스트 및 상세 데이터 연동 (결제 모듈(PG) 제외, 필요시 별도 협의)

### B. 관리자 페이지 (Admin Dashboard) 구축

- **대시보드:** 일간 방문자, 신규 회원, 수강 신청 건수 요약 통계
- **회원 관리:** 가입자 목록 조회, 등급 변경, 탈퇴 처리
- **수강 신청 관리:** 인입된 수강 신청서 목록 확인, 상태 변경(대기/상담완료/수등록 등), 메모 기능
- **콘텐츠 관리 (CMS):**
  - **교육과정 관리:** 신규 과정 등록 및 수정
  - **게시판 관리:** 공지사항 작성, Q&A 답변, 스팸 게시물/댓글 삭제
  - **갤러리 관리:** 이미지 업로드 및 카테고리 지정
  - **스토어 관리:** 상품 등록/수정/삭제, 재고 및 진열 상태 관리

### C. 백엔드 시스템 및 인프라 구축

- DB 설계 및 구축 (회원, 게시글, 수강신청, 상품 등)
- 데이터 처리를 위한 REST API
- AWS S3 스토리지 세팅 및 연동 (이미지, 첨부파일용)
- Vercel 또는 AWS를 활용한 프로덕션 배포 세팅

## 3. 세부 일정 (Total: 6주)

- **1주차:** 요구사항 최종 확정, DB 스키마 설계 및 클라우드(DB, S3) 세팅
- **2주차:** 백엔드 핵심 API 개발 및 사용자 인증(Auth) 구현
- **3~4주차:** 상세 API 구현 및 기존 퍼블리싱 코드 데이터 연동 (게시판, 갤러리, 수강신청 등)
- **5주차:** 통합 관리자 페이지(Admin) UI 개발 및 API 연동
- **6주차:** 통합 테스트, 버그 수정, 1차 피드백 반영 및 실서버 배포

## 5. 참고 및 협의 사항

1. **PG(결제) 연동:** 이번 스토어는 상품 나열 및 단순 장바구니/DB 저장 위주로 산정되었습니다. 토스페이먼츠, 이니시스 등 **실 결제 시스템 도입**이 필요할 경우 추가 공수(약 1주~2주)와 비용이 발생합니다.
2. **소셜 로그인:** 카카오, 네이버 등 간편 로그인 추가 시 사전 협의가 필요합니다.
3. **디자인 소스:** 관리자 페이지의 경우 별도의 맞춤 디자인 없이, 유지보수와 기능 중심의 깔끔한 템플릿(UI 라이브러리)을 기반으로 빠르게 개발되어 비용이 절감되었습니다.
4. **유지보수:** 런칭 후 1개월은 버그성 장애에 한해 무상 지원되며, 이후 신규 기능 추가나 디자인 변경은 별도의 유지보수 계약을 통해 진행됩니다.

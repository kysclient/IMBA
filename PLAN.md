# 국제메디컬뷰티협회 웹사이트 구현 계획

## 개요
- **사이트명:** 국제메디컬뷰티협회 (International Medical Beauty Association)
- **성격:** 협회 소개 + 교육/자격증 신청 + 커뮤니티
- **디자인 톤:** 모던 미니멀 (구글 느낌) - 깔끔한 여백, 심플한 타이포, 화이트 베이스
- **프로필 이미지:** public/profile/1.jpg, 2.jpg, 3.jpg 활용

---

## 네비게이션 구조 (KTWA 참고 + 현대화)

```
[로고] 국제메디컬뷰티협회          [협회소개] [교육과정] [갤러리] [커뮤니티] [스토어]    [로그인/회원가입]
```

### 메뉴 구성:
1. **홈 (/)** - 히어로 + 협회소개 요약 + 임원진 프로필 카드 + CTA
2. **협회소개 (/about)** - 협회 미션, 브랜드스토리, 임원진 상세, 연혁, 오시는길
3. **교육과정 (/courses)** - 커리큘럼, 자격증 정보, 가격표, 교육 신청
4. **갤러리 (/gallery)** - 교육활동, 협회행사, 미용대회 포토/영상
5. **커뮤니티 (/community)** - 공지사항, Q&A, 후기 게시판
6. **스토어 (/store)** - 교육세트, 제품 판매 (placeholder)

## 페이지별 구현 상세

### 1. 메인 페이지 (/)
- **Hero Section:** 풀폭 그라데이션 배경 + 협회 슬로건 + "교육 신청하기" CTA 버튼
- **About Preview:** 협회 한 줄 소개 + "더 알아보기" 링크
- **Team Section:** 3인 프로필 카드 (1.jpg, 2.jpg, 3.jpg) - 이름, 직책, 한줄소개
- **Services Grid:** 4개 핵심 서비스 카드 (아이콘 + 제목 + 설명)
- **Stats Bar:** 숫자 카운터 (교육생 수, 자격증 발급 수, 협력기관 등)
- **CTA Section:** 교육 상담 신청 유도

### 2. 협회소개 (/about)
- 미션/비전
- 임원진 프로필 상세 (확대 사진 + 경력 + 자격)
- 연혁 타임라인
- 오시는길 (지도 placeholder)

### 3. 교육과정 (/courses)
- 과정별 탭 (피부, 왁싱, 네일, 메이크업 등)
- 가격표 테이블
- 교육 신청 폼

### 4. 갤러리 (/gallery)
- 카테고리별 필터 (교육활동, 협회행사, 대회)
- 이미지 그리드 + 라이트박스

### 5. 커뮤니티 (/community)
- 탭: 공지사항 / Q&A / 후기
- 게시물 리스트 뷰

### 6. 스토어 (/store)
- 상품 카드 그리드 (placeholder)

## 이번 구현 범위 (Phase 1 - 프론트엔드 구조)

### 실제 구현할 것:
1. **globals.css** - 디자인 시스템 (컬러, 타이포, 간격)
2. **layout.tsx** - 루트 레이아웃 + Navbar + Footer
3. **components/Navbar.tsx** - 반응형 네비게이션 (데스크탑 + 모바일 햄버거)
4. **components/Footer.tsx** - 푸터
5. **page.tsx (홈)** - 히어로 + 협회소개 + 팀 프로필 + 서비스 + 통계 + CTA
6. **about/page.tsx** - 협회소개 페이지
7. **courses/page.tsx** - 교육과정 + 가격표 페이지
8. **gallery/page.tsx** - 갤러리 페이지
9. **community/page.tsx** - 커뮤니티 페이지
10. **store/page.tsx** - 스토어 페이지
11. **login/page.tsx** - 로그인 페이지
12. **signup/page.tsx** - 회원가입 페이지

### 디자인 원칙:
- 구글 Material Design 영감의 클린 UI
- 화이트 베이스 + 딥네이비/골드 포인트
- 넉넉한 여백, 큰 타이포
- 부드러운 hover/transition 효과
- 모바일 퍼스트 반응형
